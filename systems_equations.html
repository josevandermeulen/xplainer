Introduction to Systems of Equations

A system of linear equations is a collection of two or more linear equations involving the same set of variables. The goal is to find a common solution (a set of values for the variables) that satisfies all equations in the system simultaneously.

Examples

A common example is a system of two equations with two variables (often denoted as x and y):


  a₁x + b₁y = c₁
  a₂x + b₂y = c₂

Systems can involve more equations and variables, such as a 3x3 system:


  a₁x + b₁y + c₁z = d₁
  a₂x + b₂y + c₂z = d₂
  a₃x + b₃y + c₃z = d₃

Types of Solutions

When solving a system of linear equations, one of three outcomes is possible:

  • Unique Solution: There is exactly one set of values for the variables that satisfies all equations. Geometrically, this is where the lines (for 2D systems) or planes (for 3D systems) intersect at a single point.
  • No Solution: There are no values for the variables that can satisfy all equations simultaneously. Geometrically, this means lines are parallel and distinct, or planes do not share any common intersection point.
  • Infinitely Many Solutions: There is an infinite number of sets of values for the variables that satisfy all equations. Geometrically, this occurs when equations represent the same line (for 2D systems) or when planes intersect along a line or are coincident (for 3D systems).

Gaussian Elimination Method

Gaussian elimination is a systematic method for solving systems of linear equations. It involves transforming the system's augmented matrix into a simpler form (row echelon form or reduced row echelon form) using elementary row operations.

Augmented Matrix

First, we represent the system of equations as an augmented matrix. For a system like:


  x + 2y = 5
  3x - y = 1

The augmented matrix is:


  [ 1  2 | 5 ]
  [ 3 -1 | 1 ]

Elementary Row Operations

The following operations can be performed on the rows of the augmented matrix without changing the solution set of the system:

  • Swap: Swap two rows (Rᵢ ↔ Rⱼ).
  • Scale: Multiply a row by a non-zero constant (kRᵢ → Rᵢ).
  • Add: Add a multiple of one row to another row (Rᵢ + kRⱼ → Rᵢ).

The Goal: Row Echelon Form

The objective is to use these row operations to transform the matrix into row echelon form, which typically has ones on the diagonal (for leading coefficients) and zeros below the diagonal. From this form, the solution can be found using back substitution. Further reduction can lead to reduced row echelon form, where solutions can be read off directly.

Example: Solving the 2x2 System

Starting with our augmented matrix:


  [ 1  2 | 5 ]
  [ 3 -1 | 1 ]

Step 1: Eliminate the '3' in the second row, first column. We can do this with the operation R₂ - 3R₁ → R₂.


  [ 1  2 |  5 ]
  [ 0 -7 | -14 ]

Step 2: Make the leading coefficient in the second row '1'. We can do this with R₂ / -7 → R₂.


  [ 1  2 | 5 ]
  [ 0  1 | 2 ]

From this row echelon form, we can see that y = 2. Substituting this into the first equation (x + 2y = 5):

x + 2(2) = 5 => x + 4 = 5 => x = 1.

So the solution is (x=1, y=2).
