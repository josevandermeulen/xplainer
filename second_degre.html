Équations Linéaires (y = ax + b)

Qu'est-ce qu'une équation du second degré ?

Une équation du second degré est une équation polynomiale de degré 2, qui peut s'écrire sous la forme canonique : ax² + bx + c = 0, où 'a', 'b', et 'c' sont des coefficients réels, et 'a' est non nul.

La représentation graphique d'une fonction du second degré (y = ax² + bx + c) est une parabole. L'orientation de la parabole dépend du signe de 'a' :

  • Si 'a' > 0, la parabole est ouverte vers le haut.
  • Si 'a' < 0, la parabole est ouverte vers le bas.

Le discriminant (Δ = b² - 4ac) joue un rôle crucial pour déterminer le nombre de solutions (ou racines) de l'équation :

  • Si Δ > 0, il y a deux racines réelles distinctes.
  • Si Δ = 0, il y a une racine réelle double.
  • Si Δ < 0, il n'y a pas de racines réelles (mais deux racines complexes conjuguées).
